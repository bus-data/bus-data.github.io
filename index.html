Dataset for Delhi buses

The dynamic data for three months has been stored separately and can be downloaded using the link below. This data is currently available for two routes namely - 534 and 425. This has been sampled at every 10s using the real-time data for these routes. The data provides the geographical location of the buses along with the time stamp. This data can be used for various purposes including GPS tracked monitoring of traffic, tracking anomalies in live traffic data for security purposes, travel time estimation and optimization.
